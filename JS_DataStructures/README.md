```js
<head>
	<meta charset="utf-8">
	<link rel="stylesheet" href="./styles.css">
	<title>RPG - Dragon Repeller</title>
</head>

<body>
	<div id="game">
	</div>
</body>

</html>
```
2. 在网页端输入以上代码并提交；
